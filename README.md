# Healthcare Authentication System

A full-stack healthcare authentication platform with patient and doctor modules.

## Features
- Patient Signup & Login
- Doctor Registration with Approval Logic
- Patient Dashboard
- Doctor Dashboard
- File Upload (Photo & Certificate)
- Secure Backend APIs

## Tech Stack
Frontend:
- React.js
- Axios
- React Router

Backend:
- Node.js
- Express.js
- Multer (File Upload)
- Nodemailer (Email Ready)

## Modules
- Patient Authentication
- Doctor Authentication
- Dashboards
- File Upload System

## Run Locally

### Backend
cd backend  
npm install  
node server.js  

### Frontend
cd frontend  
npm install  
npm run dev
